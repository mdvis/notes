| 功能描述          | APT/dpkg                  | DNF/RPM                        | Pacman                           |
| :------------ | :------------------------ | :----------------------------- | :------------------------------- |
| **核心工具类型**    | `dpkg` (低级), `apt` (高级)   | `rpm` (低级), `dnf` (高级)         | `pacman` (兼具高低级)                 |
| **安装本地包**     | `sudo dpkg -i pkg.deb`    | `sudo rpm -ivh pkg.rpm`        | `sudo pacman -U pkg.pkg.tar.zst` |
| **安装仓库包**     | `sudo apt install pkg`    | `sudo dnf install pkg`         | `sudo pacman -S pkg`             |
| **卸载包**       | `sudo apt remove pkg`     | `sudo dnf remove pkg`          | `sudo pacman -R pkg`             |
| **卸载包及依赖**    | `sudo apt autoremove pkg` | `sudo dnf autoremove pkg`      | `sudo pacman -Rs pkg`            |
| **更新软件源列表**   | `sudo apt update`         | `sudo dnf check-update`        | `sudo pacman -Sy`                |
| **升级所有软件**    | `sudo apt upgrade`        | `sudo dnf upgrade`             | `sudo pacman -Syu`               |
| **搜索软件包**     | `apt search pattern`      | `dnf search pattern`           | `pacman -Ss pattern`             |
| **查看包信息**     | `apt show pkg`            | `dnf info pkg`                 | `pacman -Si pkg`                 |
| **列出已安装包**    | `dpkg -l`                 | `rpm -qa` `dnf list installed` | `pacman -Q`                      |
| **查询文件属于哪个包** | `dpkg -S /path/to/file`   | `rpm -qf /path/to/file`        | `pacman -Qo /path/to/file`       |
| **列出包内文件**    | `dpkg -L pkg`             | `rpm -ql pkg`                  | `pacman -Ql pkg`                 |
| **清理包缓存**     | `sudo apt clean`          | `sudo dnf clean all`           | `sudo pacman -Sc`                |
| **查找孤儿包**     | `deborphan` (需安装)         | `sudo dnf autoremove`          | `pacman -Qdtq`                   |
