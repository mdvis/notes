根据Sioyek官方文档的内容，Sioyek 的设计初衷是为了高效阅读学术论文和技术书籍，其操作主要依赖于快捷键（类似Vim）。所有的默认按键都可以通过修改 `keys_user.config` 文件来自定义，你随时可以按 `:` 打开所有可用命令的搜索列表。

以下是 Sioyek 核心操作与快捷键的总结：

### 1. 文件管理 (Opening Files)

* **`o`**: 打开文件选择菜单。
* **`Shift + o` (O)**: 搜索并打开最近浏览过的文件。
* **`Ctrl + o`**: 打开内嵌的文件系统浏览器。

### 2. 基础移动与导航 (Basic Movement)

* **方向键 / 鼠标滚轮**: 移动页面。
* **`Space` / `Shift + Space**`: 向下 / 向上滚动屏幕。
* **`Ctrl + PageDown` / `Ctrl + PageUp**`: 下一页 / 上一页。
* **`gg` / `G**`: 跳转到第一页 / 最后一页（输入数字加 `gg` 可以跳转到特定页，例如 `42gg` 跳转到第42页）。
* **`t`**: 打开并搜索目录（TOC）。
* **`gc` / `gC**`: 跳转到下一章 / 上一章。

### 3. 缩放与视图 (Zoom & Window)

* **`+` / `-**`: 放大 / 缩小（也可按住 `Ctrl` 加上鼠标滚轮）。
* **`f9`**: 宽度适应窗口。
* **`f10`**: 智能宽度适应（忽略白边）。
* **`f8`**: 切换深色模式。

### 4. 历史跳转 (History Navigation)

* **`Backspace` / `Shift + Backspace**`（或 `Ctrl+左/右方向键`）: 在浏览历史记录中后退 / 前进。该功能支持**跨文档**使用，极大地丰富了论文之间的交叉阅读。

### 5. 悬浮窗与智能跳转 (Overview and SmartJump)

* **右键点击**: 在PDF链接或引用文字（如“Figure 2.19”）上右键，会打开一个快速**悬浮窗**预览目标内容，支持鼠标滚轮在悬浮窗内滚动，无需离开当前阅读位置。
* **中键点击**: 直接跳转到引用目标的位置。

### 6. 标记与高亮 (Marks & Highlights)

* **视觉标记 (Visual Mark)**: 在某行文字右键，会在下方生成一条高亮线，帮助你防跟丢。按 `j` / `k` 可上下移动这根线。
* **位置标记 (Marks)**: 类似于Vim，按 **`m`** 加上 **一个字母**（如 `ma`）记录当前位置，按 **```** 加上 **对应字母**（如 ``a`）快速跳回。小写字母限当前文档，大写字母跨文档全局有效。
* **书签 (Bookmarks)**: 按 **`b`** 添加带文本描述的书签；按 **`gb`** / **`gB`** 搜索当前/全部文档的书签。
* **文本高亮 (Highlights)**: 选中文本后按 **`h`** 加上 **一个字母**，即可添加对应颜色类型的高亮；按 **`gh`** 可搜索高亮。

### 7. 搜索 (Search)

* **`/` 或 `Ctrl + f**`: 打开搜索菜单。
* **`n` / `N**`: 跳转到下一个 / 上一个搜索结果。
* **`c/`**: 仅在当前章节内进行搜索。
* **范围搜索**: 可以限定页码，如 `<20,30>Figure` 表示在第20至30页中搜索“Figure”。
* **外部搜索**: 选中文本后按 **`ss`** 或 **`sl`** 可以在 Google Scholar 或 Library Genesis 中快速搜索。中键点击论文名称也能自动触发外部搜索。

### 8. 传送门 (Portals - 核心特色功能)

当你在阅读一段解释图表或公式的文字，而图表在其他页面时，来回翻页会非常痛苦。

* **`p`**: 按 `p` 标记当前文字位置，然后跳转到图表所在的位置，再按一次 `p` 即可建立连接。以后当你浏览到这段文字时，关联的图表会自动在副窗口（按 `f12` 切换辅助窗口）中显示出来。

### 9. 其他进阶操作

* **`f4`**: 切换 Synctex 模式（用于与 LaTeX 源码进行正反向搜索交互）。
* **`f5`**: 切换到演示模式 (Presentation mode)。
* **`:`**: 打开命令菜单，可以在此执行所有未绑定快捷键的高级操作。
