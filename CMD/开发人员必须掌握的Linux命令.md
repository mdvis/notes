## 💡 学习建议

### 1. 学习路径
1. **先掌握级别1的所有命令** - 覆盖90%的日常开发场景
2. **重点练习管道操作** - 如 `grep "error" logfile.log | wc -l`
3. **熟练使用Git** - 现代开发的标配
4. **学会查阅`man`手册** - `man command` 了解详细用法

### 2. 实践技巧
- 多使用Tab键自动补全
- 使用上下箭头查看历史命令
- 学会使用`alias`创建快捷命令
- 结合`|`（管道）和`>`（重定向）使用命令

### 3. 常用组合
```bash
# 查找包含error的行数
grep "error" logfile.log | wc -l

# 批量重命名文件
for file in *.txt; do mv "$file" "${file%.txt}.bak"; done

# 查找最近修改的文件
find . -name "*.js" -mtime -1

# 查看大文件
du -sh * | sort -hr | head -10
```
