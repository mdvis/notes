# Linux 命令学习指南 (DevOps / 全栈开发版)

这份指南专为**软件开发人员（兼顾运维/DevOps 职责）**整理。除了日常开发必备命令外，重点补充了系统监控、故障排查、网络配置和磁盘管理等 Ops 技能。

---

## 🟢 第一梯队：核心高频 (生存必备)
**无论是开发还是运维，这些是每天都要打交道的“肌肉记忆”级命令。**

### 1. 开发与部署
- `git`: 版本控制核心。
- `docker`: 容器化应用与环境。
- `make`: 自动化构建与任务执行。
- `systemctl`: **(Ops)** 管理系统服务（启动/停止/重启 Nginx, MySQL, Docker 等）。

### 2. 系统状态与监控
- `ps`: 查看进程状态 (`ps aux | grep java`).
- `top`: 实时监控 CPU/内存负载。
- `free`: 查看内存使用情况 (`free -h`).
- `df`: 查看磁盘空间占用 (`df -h`).
- `du`: 查看目录或文件大小 (`du -sh *` 排查谁占用了磁盘).

### 3. 文件操作与导航
- `ls`, `cd`, `pwd`: 基础导航。
- `cp`, `mv`, `rm`: 文件操作 (注意 `rm -rf` 的风险).
- `mkdir`, `touch`: 创建目录和文件。
- `cat`, `less`, `tail`, `head`: 查看文件内容。`tail -f` 必会，用于实时看日志。

### 4. 网络基础
- `curl`: 测试接口、下载文件、检查服务存活。
- `ping`: 测试连通性。
- `ip`: 查看网卡与 IP 信息 (替代 old `ifconfig`).
- `lsof`: **(Ops)** 查看端口占用 (`lsof -i :80`) 或文件占用。

### 5. 权限管理
- `chmod`: 修改读写执行权限。
- `chown`: 修改文件所有者 (部署/挂载卷常用).
- `sudo`: 提权操作。

---

## 🟡 第二梯队：运维排查与脚本工具 (进阶效率)
**当你需要排查线上故障、编写自动化脚本或进行服务器维护时，这些命令不可或缺。**

### 1. 文本流处理 (日志分析利器)
- `grep`: 强大的文本搜索。
- `awk`: 列提取与统计 (如从日志中提取 IP).
- `sed`: 文本替换与编辑 (修改配置文件).
- `find`: 精确查找文件 (按时间、大小、名称).
- `sort`, `uniq`, `wc`: 排序、去重、统计行数 (分析访问日志常用组合).
- `xargs`: 参数传递工具 (配合 find 删除旧日志等).
- `diff`: 对比配置文件差异.

### 2. 网络故障排查
- `dig`, `host`: DNS 解析诊断.
- `iptables`, `firewall-cmd`: **(Ops)** 防火墙规则管理 (端口放行/拦截).
- `ifconfig`: 老牌网络配置工具 (部分旧系统仍需).

### 3. 定时任务与后台
- `crontab` / `cron`: 管理定时任务 (备份、清理).
- `bg`, `fg`, `jobs`: 管理后台任务.

### 4. 压缩与归档
- `tar`: 万能打包解压 (`zcvf`, `zxvf`).
- `zip`, `unzip`: 处理跨平台的 zip 包.

### 5. 环境与安全
- `env`, `export`: 环境变量管理.
- `history`: 查找操作历史.
- `openssl`: 证书与 SSL 调试.
- `md5sum`, `sha256sum`: 文件完整性校验.

---

## 🔵 第三梯队：底层硬件与磁盘 (Ops 专项)
**涉及服务器硬件配置、磁盘扩容、挂载等低频但关键的运维操作。**

### 1. 磁盘与文件系统
- `mount`, `umount`: 挂载/卸载磁盘.
- `fdisk`, `lsblk`, `blkid`: 磁盘分区查看与管理 (新硬盘挂载).
- `mkfs`: 格式化磁盘.
- `fsck`, `e2fsck`: 文件系统修复 (系统起不来时救命用).
- `hdparm`: 硬盘参数与性能测试.

### 2. 硬件信息
- `lscpu`: 查看 CPU 架构与核数.
- `lsmod`: 查看内核模块.
- `dmesg`: 查看内核环形缓冲区信息 (排查硬件故障/OOM killer).
- `lsb_release`: 查看 Linux 发行版版本.

---

## ⚪ 第四梯队：其他/按需查阅
**日常使用频率较低，或有更现代的替代品，只需了解功能。**

- **Shell 内部**: `alias`, `source`, `clear`, `declare`, `command`, `type`.
- **低频/老旧**: `finger`, `chsh`, `at`, `dump`, `arp` (常用 `ip neigh`), `iw`.
- **特定场景**: `dd` (制作镜像/测试磁盘写速度), `hexdump`, `base64`.
- **其他**: `date` (脚本中常用), `killall` (不如 `kill` 精准).
