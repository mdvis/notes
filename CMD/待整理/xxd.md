`xxd` 将文件转成十六进制视图（和 `hexdump` 类似），还可以反向把十六进制再转回二进制
常用选项：

| 选项          | 含义                   |
| ----------- | -------------------- |
| `-g N`      | 分组间隔（默认 2 字节一组）      |
| `-c N`      | 每行显示 N 个字节（默认 16）    |
| `-s OFFSET` | 从指定偏移开始              |
| `-l LENGTH` | 限制输出长度               |
| `-r`        | **反向操作**：从十六进制还原成二进制 |
| `-p`        | 输出纯十六进制（无地址、空格）      |
