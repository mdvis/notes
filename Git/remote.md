# remote
\[TOC\]

| 名称  | 作用  |
| --- | --- |
| git remote | 仓库短名称 |
| git remote -v,--verbose | 显示克隆地址 |
| git remote add \[shortname\] \[url\] | 关联到远程仓库（可以关联多个远程仓库） |
| git remote set-url \[shortname\] \[url\] | 修改关联地址 |
| git remote prune origin |     |
| git remote show origin | 查看远程分支信息;远程分支，地址及对应关系 |
| git remote rename \[oldname\] \[newname\] | 修改远程仓库本地简称 |
| git remote rm \[shortnaem\] | 删除远程仓库 |
