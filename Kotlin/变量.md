# 变量
数据类型

* * *

|     |     |     |
| --- | --- | --- |
| **Kotlin 数据类型** | **可包含的数据类型** | **字面量值示例** |
| `String` | 文本  | `“Search”` |
| `Int` | 整数  | `-59281` |
| `Double` | 小数  | `-23.999` |
| `Float` | 小数（不如 `Double` 精确），数字末尾带有 `f` 或 `F`。 | `1.2940278F` |
| `Boolean` | `true` 或 `false` |     |

变量
--

```text-plain
val name: <data type> = <initial value>
```

```text-x-kotlin
val count: Int = 2
```

**字符串模板**

```text-x-kotlin
println("You have $count unread message.")
```

**类型推断**

```text-x-kotlin
val name = <initial value>
```

**整数基本运算**

```text-x-kotlin
println("You have ${unreadCount + readCount} total messages in your inbox.")
```

*   val 预计变量值不会变化时使用（变量只读）
*   var 预计变量会发生变量时使用

**规范**

*   变量名使用小驼峰
*   变量声明冒号和类型之间加一个空格
*   赋值运算符 `=`、`+`、`-`、`*`、`/` 等运算符前后空格
*   每行尽量不超过 100 字符

**注释**

`//` 单行注释，`/*`和`*/` 多行注释
