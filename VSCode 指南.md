这些是基础操作，能覆盖80%的日常需求。优先掌握命令面板和快速打开文件。

| 功能          | Windows/Linux     | macOS             | 备注                          |

#### 七、独家技巧与自定义
高级用法和个性化建议，提升整体体验。

1. **查询所有快捷键**：
   - Windows/Linux: Ctrl + K, Ctrl + S（按顺序按）
   - macOS: Cmd + K, Cmd + S
   - 在此面板中，可修改任何键位。

2. **命令面板高级用法**（打开后 Ctrl/Cmd + P）：
   - 输入 `>`：进入命令模式（等同 Ctrl/Cmd + Shift + P）。
   - 输入 `:`：跳转到行号（等同 Ctrl/Cmd + G）。
   - 输入 `@`：跳转到当前文件的符号（函数、类）。
