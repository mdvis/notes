
| Name            | File Desc. | Desc. | Abbr.  |
| --------------- | ---------- | ----- | ------ |
| Standard Input  | 0          |       | stdin  |
| Standard Output | 1          |       | stdout |
| Standard Error  | 2          |       | stderr |
## 输出重定向
`File_Desc >& Pointer`

`0<` 标准输入
`1>` 标准输出，简写 `>`
`2>` 错误输出
`&>` 为 `2>&1` 缩写
