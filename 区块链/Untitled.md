# 🧭 区块链开发学习路线

## **第一阶段：打好基础**

1. **计算机基础**
    
    - 数据结构与算法（链表、哈希表、树、Merkle Tree）
        
    - 网络与分布式系统（P2P、共识、CAP 理论）
        
    - 操作系统与并发（线程、消息队列）
        
2. **密码学与安全**
    
    - 哈希函数（SHA-256, Keccak256）
        
    - 非对称加密（RSA, ECDSA）
        
    - 数字签名与零知识证明（基础概念即可）
        

📚 推荐资源：

- 《精通比特币》 (Mastering Bitcoin, Andreas Antonopoulos)
    
- Coursera: Blockchain Specialization
    

---

## **第二阶段：理解区块链原理**

1. **比特币机制**
    
    - 区块、交易、UTXO 模型
        
    - POW 共识机制
        
2. **以太坊机制**
    
    - 账户模型 vs UTXO
        
    - EVM (Ethereum Virtual Machine)
        
    - Gas, Transaction, Block
        

📚 推荐资源：

- 《精通以太坊》(Mastering Ethereum, Andreas Antonopoulos)
    
- Ethereum 官方文档
    

---

## **第三阶段：智能合约开发**

1. **Solidity 基础**
    
    - 数据类型、函数、事件
        
    - 修饰符（modifier）、存储（storage/memory）
        
    - 合约继承、库
        
2. **实践**
    
    - 实现一个 ERC-20 代币
        
    - 实现一个 NFT (ERC-721)
        
    - 部署到测试网（Goerli, Sepolia）
        
3. **开发工具链**
    
    - Remix IDE（上手最快）
        
    - Hardhat / Foundry / Truffle（专业开发）
        
    - Web3.js / Ethers.js 与前端交互
        

📚 推荐资源：

- CryptoZombies (交互式游戏式 Solidity 教程)
    
- OpenZeppelin 文档（安全合约标准库）
    

---

## **第四阶段：区块链高级主题**

1. **扩展性与 Layer 2**
    
    - Rollups (Optimistic, ZK)
        
    - Sidechains, Plasma
        
2. **跨链技术**
    
    - Polkadot, Cosmos
        
    - 桥接协议（Bridge）
        
3. **安全与审计**
    
    - 常见攻击（重入攻击、整数溢出、闪电贷攻击）
        
    - 使用 Slither、MythX 做合约审计
        

📚 推荐资源：

- Ethernaut（黑客式智能合约闯关游戏）
    
- DeFi Hack Labs / Damn Vulnerable DeFi
    

---

## **第五阶段：构建作品集（Portfolio）**

目标：有实际项目，面试更有竞争力。

🔨 项目建议：

1. 一个代币 + 简单 DApp（前端 + 合约）
    
2. NFT 市场（铸造、交易、展示）
    
3. 去中心化投票系统
    
4. 一个简单的 DeFi 应用（质押/借贷）
    

---

## **第六阶段：求职准备**

- **GitHub**：上传代码，写 README
    
- **Blog/知乎**：写学习笔记，展示对技术的理解
    
- **面试准备**：
    
    - Solidity 原理与最佳实践
        
    - 智能合约常见安全问题
        
    - 区块链基本原理
        

---

⚡ 建议学习方式：

- **2 周**打好理论和比特币/以太坊基础
    
- **1 个月**集中练习 Solidity 和 DApp
    
- **1-2 个月**做出完整的作品集项目
    
- **并行**写学习笔记、上传代码
