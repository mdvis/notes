# 醋泡黑豆   
**核心步骤**：选豆→清洗→炒制→泡醋→保存   
1. **选豆与清洗**   
    - **黑豆选择**：优先用 **绿芯黑豆**（营养更丰富），剔除杂质后洗净沥干。   
2. **炒制关键**   
    - **干炒火候**：黑豆放入无油锅，**中火炒至表皮爆裂**（约5分钟），转小火继续炒 **5-8分钟**，至豆香浓郁且冷却后不软烂。   
    - **避免焦糊**：需频繁翻动，防止局部过热。   
3. **泡醋方法**   
    - **醋的选择**：用 **陈醋或米醋**，用量需完全没过黑豆（可预留膨胀空间）。   
    - **密封保存**：炒好的黑豆晾凉后装入玻璃罐，倒入醋密封，冷藏 **3-5天** 即可食用，每日 **5-10粒**。   
4. **增效技巧**   
    - **分层泡制**：罐底放黑豆，中层加枸杞或红枣，上层淋少量蜂蜜，提升口感和营养。   
    - **二次利用**：泡完醋的黑豆可烘干磨粉，加入粥中食用。
