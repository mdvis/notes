## 惰性求值
-  不使用不计算（值、属性集元素）
```
nix-instantiate --eval *.nix
nix-instantiate --eval --strict *.nix
```
## 注释、缩进、换行
- 注释用 `#`
- 缩进大多数情况是为了可读性
- 换行和空格具有分割作用，不可随意断行
## 名称、属性集
名称可以理解成变量，更像常量不可变
- 字符串 `"string"`
- 整数 `1,2,3`
- 浮点数 `3.142,2.718`
- 布尔 `true,false`
- null `null`
- 列表 `["" 1 false]`
- 属性集 `{ a = 1; b = "tux"; c = false }`
- 函数 `x: x + 1`
### 属性集
- 键值对
- 末尾必须加`;`
```
{
	a=1;
	b={
		c=2;
	};
	b.d=3; // 属性访问
}
```
#### 递归属性集
普通属性集不支持递归引用
```
{
	a=1;
	b=a+1; # error: u
}
```
