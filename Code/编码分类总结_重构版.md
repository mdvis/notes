# 编码分类总结（重构版）

## 一、按用途分类

### 1.1 文本字符编码
#### 常用
- **UTF-8** - 现代互联网和软件开发的标准编码，兼容ASCII，支持全球所有语言
- **ASCII** - 计算机基础编码，所有现代编码的基础
- **GB2312/GBK** - 中文编码标准，在中国广泛使用

#### 重要但不常用
- **GB18030** - 中国国家标准，在政府和企业系统中将强制使用，覆盖字符最全的中文编码
- **UTF-32** - 在需要固定字符宽度的国际化处理中可能变得重要，适用于复杂的多语言文本处理算法

#### 趋势
- **GB18030** - 中国国家标准，在政府和企业系统中将强制使用，覆盖字符最全的中文编码
- **UTF-32** - 在需要固定字符宽度的国际化处理中可能变得重要，适用于复杂的多语言文本处理算法

### 1.2 二进制传输编码
#### 常用
- **Base64** - 最常见的二进制到文本的编码方式，用于邮件、数据URL等
- **URL编码** - 网页和HTTP请求中传输特殊字符的标准方式

#### 重要但不常用
- **Base58Check** - 比特币地址等加密货币应用的标准编码，防止输入错误，提高安全性

#### 趋势
- **Base58Check** - 随着区块链和加密货币技术的发展而变得更加重要

### 1.3 数据序列化格式
#### 常用
- **JSON** - Web API和配置文件的事实标准，易于阅读和解析
- **XML** - 在企业级应用和文档存储中仍广泛使用
- **YAML** - 配置文件的流行选择，可读性强

#### 重要但不常用的
- **Protobuf (Protocol Buffers)** - 高性能RPC通信、微服务间数据交换的事实标准，体积小、解析速度快、支持多语言
- **Avro** - Apache Kafka、Hadoop生态系统的核心数据格式，支持模式演化，非常适合大数据处理场景
- **MessagePack** - 二进制JSON的高效替代方案，比JSON更紧凑，解析速度更快

#### 趋势
- **YAML** - 在DevOps、容器编排（如Kubernetes、Docker Compose）中越来越重要，随着云原生技术发展将成为基础设施即代码的标准

### 1.4 多媒体编码
#### 常用
- **JPEG** - 数码照片和网络图片的主要格式
- **PNG** - 需要透明背景的图像和网页图标的首选
- **MP3** - 音频压缩的事实标准
- **H.264** - 视频流媒体和存储的主流编码

#### 重要但不常用的
- **FLAC (无损音频)** - 音乐存档、高保真音频播放的首选，无损压缩，音质完美
- **HEVC (H.265) 和 VP9** - 4K/8K超高清视频的未来标准，相比H.264压缩效率提高一倍，节省50%带宽

#### 趋势
- **AV1 (AOMedia Video 1)** - Google、Mozilla、Netflix等支持的开源免版税视频编码，有望成为下一代视频编码标准，取代H.265/VP9
- **HEVC (H.265)** - 随着硬件支持完善，将在4K/8K视频时代发挥更大作用

### 1.5 文件魔数
#### 常用
用于快速识别文件类型
- JPEG 图像: FF D8
- PNG 图像: 89 50 4E 47 0D 0A 1A 0A
- GIF 图像: 47 49 46 38 37 61 (GIF87a) 或 47 49 46 38 39 61 (GIF89a)
- BMP 图像: 42 4D
- PDF 文档: 25 50 44 46
- ZIP 压缩包: 50 4B 03 04
- Word 文档 (旧): D0 CF 11 E0 A1 B1 1A E1
- Excel 表格 (旧): D0 CF 11 E0 A1 B1 1A E1
- Word 文档 (新): 50 4B 03 04
- Excel 表格 (新): 50 4B 03 04
- RAR 压缩包: 52 61 72 21 1A 07 00
- 7-Zip 压缩包: 37 7A BC AF 27 1C 00 03
- MP3 音频: 49 44 33
- MP4 视频: 00 00 00 18 66 74 79 70
- AVI 视频: 52 49 46 46
- WAV 音频: 52 49 46 46
- RTF 文本: 7B 5C 72 74 66 31

### 1.6 通信线路编码
#### 常用
- **HDB3码** - 在电信系统中用于保证数据同步

## 二、总结

| 用途      | 常用编码                                      | 重要但不常用编码                    | 趋势编码             |
| ------- | ----------------------------------------- | --------------------------- | ---------------- |
| 文本字符编码  | UTF-8, ASCII, GB2312/GBK                  | GB18030, UTF-32             | GB18030, UTF-32  |
| 二进制传输编码 | Base64, URL编码                             | Base58Check                 | Base58Check      |
| 数据序列化格式 | JSON, XML, YAML                           | Protobuf, Avro, MessagePack | YAML             |
| 多媒体编码   | JPEG, PNG, MP3, H.264                     | FLAC, HEVC(H.265), VP9      | AV1, HEVC(H.265) |
| 文件魔数    | JPEG(FF D8), PNG(89 50...), ZIP(50 4B...) | -                           | -                |
| 通信线路编码  | HDB3码                                     | -                           | -                |
