- **更新软件包列表并升级所有软件包**：
```bash
sudo dnf update
```
此命令会更新软件包列表并将所有可升级的软件包升级到最新版本。
- **安装软件包**：
```bash
sudo dnf install <package_name>
```
使用软件包名称替换 `<package_name>` 来安装软件。
- **移除软件包**：
```bash
sudo dnf remove <package_name>
```
此命令用于移除指定的软件包。
- **查找软件包**：
```bash
sudo dnf search <keyword>
```
用关键词替换 `<keyword>` 来查找包含该关键词的软件包。
