## 基本类型（8个
- 整型
	1. int         4b    -2,147,483,648 ~ 2,147,483,647
	2. short.   2b     -32,768 ~ 32,767
	3. long.     8b    -9,223,372,036,854,775,808 ~  9,223,372,036,854,775,807
	4. byte.     1b     -128 ~ -127

```
10       // int
4000000l // long
4000000L // long
010      // 8进制
0x10     // 16进制
0b1001   // 2进制
0B1001   // 2进制
100_100  // 下划线可以做分割，编译器会去除
```
